# Tugas Besar Pemrograman Mobile

Project ini adalah sebuah aplikasi dimana user dapat melihat data profile karakter-karakter dari
permainan mobile bernama Colorful Stage: Hatsune Miku atau dalam versi Jepangnya yang bernama
Project Sekai: Colorful Stage ft. Hatsune Miku. Project ini dibuat untuk menyelesaikan tugas besar
mata kuliah Pemrograman Mobile yang ditugaskan oleh Bapak Robby Tan selaku Dosen kami.
